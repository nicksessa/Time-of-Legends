# Arthurian Style Weapons, Armor, Items

| Roll | Weapon                                                                                                                                                                                       |
| ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|      | Balin's Broadsword - This weapon gives a +1 to ATT and DEF.  If the user rolls a 6 to hit, instead of exploding, the wielder can instead make a second attack roll.                          |
|      | Sir Bor's Sword - A Sacred Weapon (Kills on all doubles) that can fully heal the bearer once per day.                                                                                        |
|      | Galatine - Once per day the wielder of this sword can bring forth a blinding flash of light that dazzles all opponents within Long range.  All who fail a 4+ save are blinded for one round. |
|      | Relic Sword - A blessed sword with the relic of a saint in the pommel or crosspiece. Demons and dragons wounded by a relic sword read one line lower on the wound chart.                     |
|      | Perceval's Sword - This fine sword ignores all armor, but not Toughness!                                                                                                                     |
|      | Priamus' Sword - Anyone wounded by this sword loses one point of ATT.  If ATT goes to -1, that model is killed.                                                                                                                                                                                             |


| Roll | Armor                                                                                                                                                                                                      |
| ---- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|      | The Adventurous Shield - This medium shield was once owned by Galahad and provides +1 to DEF.  Additionally, the wielder can ignore the first wound roll made against him in a combat encounter.           |
|      | Blessed Armor - Wearers of Blessed Armor suffer no movement penalties due to wearing heavy armor.                                                                                                          |
|      | The Golden Harness - The wearer of the Golden Harness (DR 6) gains a +1 to Leadership.  All allies withing long range of the wearer of this armor also may reroll failed leadership tests once per battle. |
|      | Viviane's Shield - These medium sized silver shields were given to Lancelot by the Lady of the Lake.  Each shield has three horizontal red stripes on them that allow the wielder to ignore three wounds.  Once the shield has protected it's wielder three times, the shield loses it's magical properties.                                                                                                                                                                                                           |
